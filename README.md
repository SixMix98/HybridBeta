# Hybrid Beta

This mod is designed to modify the beta 1.7.3 experience by
implementing several old and new features, along with
numerous tweaks and fixes.  You can check the wiki for 
a full list of features


## How To Install on MultiMC

Most launchers should support this mod, though so far
it has only been tested on MultiMC.


1. Download Babric.  You most likely will want to select the PrismLauncher/MultiMC option (https://babric.github.io/use/installer/)
2. Open your launcher.  In the case of MultiMC, I can just drop the instance zip file into it.
3. Add StationAPI to your mod folder (In MultiMC this is done with Edit Instance>Loader Mods>Add>StationAPI[version].jar)
4. If you haven't already downloaded a jar file for this mod, go to releases and select the latest version.  Add it to your mod folder as well.
5. Launch the game and have fun!

StationAPI Download:
https://modrinth.com/mod/stationapi

If you followed the instructions and are still having issues, 
check to make sure you are using Java 17 or above.
